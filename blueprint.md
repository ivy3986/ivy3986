# Threads Content Generator for Coupang Partners

## Overview
A specialized tool for affiliate marketers to generate high-engagement content for Threads. It processes video uploads and generates content in a trendy, conversational style.

## Project Outline
- **Target Platform:** Threads (optimized for short, engaging text).
- **Core Feature:** Video Upload -> AI Analysis -> MZ-style Description -> Coupang Link.
- **Visual Design:** Glassmorphism UI.
